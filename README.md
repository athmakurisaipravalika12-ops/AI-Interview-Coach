# AI-Interview-Coach
AI Interview Coach is an AI-powered platform that helps students and job seekers prepare for interviews through mock interview sessions, topic-based questions, automated answer evaluation, and personalized feedback. It improves confidence, identifies skill gaps, and provides a realistic interview practice experience.

## Problem Statement
Many students struggle to prepare for technical interviews due to lack of practice and personalized feedback.

## Solution
AI Interview Coach conducts mock interviews and provides feedback on user answers.

## Features
- Mock interview sessions
- Python, SQL, and Flask questions
- AI-based feedback
- Performance scoring
- Progress tracking

## Technology Stack
- Python
- Flask
- SQLite
- AI API

## Workflow

Student
↓
Select Topic
↓
Answer Questions
↓
AI Evaluation
↓
Score & Feedback

## Future Scope
- Voice interviews
